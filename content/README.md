# Content Management System

This directory contains the content management system for the Stone OnePoint Solutions website. It provides tools and templates for easy content maintenance, validation, and versioning.

## Directory Structure

```
content/
├── templates/          # Content templates for consistent formatting
├── data/              # Structured content data files
├── scripts/           # Content management scripts
├── backups/           # Content backup and versioning
└── validation/        # Content validation tools
```

## Quick Start

1. **Update Content**: Use templates in `templates/` directory
2. **Validate Content**: Run `npm run validate-content`
3. **Backup Content**: Run `npm run backup-content`
4. **Deploy Changes**: Run `npm run deploy-content`

## Content Guidelines

- Use provided templates for consistency
- Validate content before deployment
- Always backup before major changes
- Follow SEO best practices in templates
